# Simulador de Indução Eletromagnética

Simulador interativo desenvolvido para o currículo de Física de 11.º ano (Portugal), focado nos conceitos de fluxo magnético, Lei de Faraday e Lei de Lenz.

## Funcionalidades
- **Visualização 2D Lateral**: Representação clara da espira e do versor normal.
- **Tipos de Campo**: Suporte para Campo Uniforme e Íman em Barra (interativo).
- **Gráficos em Tempo Real**: Visualização simultânea de Fluxo Magnético (Φ) e Força Eletromotriz (ε).
- **Assistente de Física (Chatbot)**: IA heurística para explicar conceitos e responder a dúvidas sobre a simulação.
- **Controles Avançados**: Ajuste de área, velocidade angular e número de espiras.

## Como correr
Basta abrir o ficheiro `index.html` em qualquer navegador moderno.

Alternativamente, se estiver num ambiente Unix:
```bash
./run.sh
```

## Atalhos de Teclado
- **Espaço**: Play / Pause
- **Setas ← →**: Ajuste manual do ângulo
- **R**: Reiniciar simulação
